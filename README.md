# Web3 Scout

CLI tool for Web3 project analysis and multi-platform marketing automation.

## Features

- **Project Analysis**: Analyze Web3 project credibility with scoring system
  - GitHub activity metrics
  - Social media presence
  - Token contract data
  
- **Multi-Platform Publisher**: Publish content to 30+ platforms
  - Twitter/X, Telegram, Discord
  - Reddit, Medium, Mirror
  - Lens, Farcaster, Mastodon, Bluesky
  - And 20+ more Web3 and social platforms
  
- **Campaign Tracking**: Dashboard showing post performance across all platforms
